# Unichat Application


### Introduction
This is a full Realtime Chat app with Social Auth and dedicated chat APIs/sockets created using React.js .
<aside>MVP</aside>

### Features 
- Realtime chatting 
- Signing up using facebook
- Signing up using google
- Status while chatting
